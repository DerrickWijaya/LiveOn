<p>Copy the <code>.env.example</code> file, rename it to <code>.env</code>, and create your own database.</p>
<p>Since there is no <code>vendor</code> folder, run <code>composer install</code> in your terminal to generate it.</p>
<p>Run <code>npm install</code> and <code>npm run build</code>.</p>
<p>Your website is now ready on your local environment.</p>
